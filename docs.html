buy(self, symbol, quantity, in_force, extended)

Buys the specified asset

Params:

Default

• symbol

 (str):

Ticker symbol of the asset to buy.

First symbol in 'watch'

• quantity

 (float):

Quantity of asset to buy.

1

• in_force

 (str):

Duration the order is in force. 'gtd' or 'gtc'

gtd

• extended

 (str):

Whether to trade in extended hours or not

False

Returns:

A dictionary with the following keys: - type: 'STOCK' or 'CRYPTO' - id: ID of order - symbol: symbol of asset

Raises:

Raises an exception if order fails.
